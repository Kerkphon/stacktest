Hello World Tutorial!

In this tutorial, we will build a simple web application using part of the MEAN stack. We'll run the "Hello World" application on Google Cloud using Node.js and Express.js! This includes creating a Virtual Machine, installing git, nodejs, and npm. The Github icon is on the top right of the page. This will take you to the repository to clone/fork the repository if you're feeling adventurous. Be sure the filesâ€™ names are correct and call the right PORT name on your VM. Ready to make the first application? It begins right here!


Create VM Instance

1. Create your instance name to: helloworld-vm


2. Boot disk: Google Debian GNU/Linux(default)


3. Machine type: changed to small(1 shared vCPU, 1.7 GB memory)


4. Check both HTTP & HTTPS boxes then create


5. Go into Networking > VPC Network > Firewall


6. Edit the rules for "default-allow-http" & "default-allow-https" adding port number 3000


Install Node

1. First get all the new update for your VM

                                sudo apt-get update


2. Install Nodejs

                                sudo apt-get install -y nodejs


3. Install Node Package Management

                                sudo apt-get install npm


4. Check if Node is running 

                                node -v


5. Check if Node Package Management is running

                                npm -v


Install Express

1. Initialize a package.json file with the following command

                                     sudo npm init


2. Add a start script to your package.json file

                                    "script":{ 
"start": "app.js"


3. Install Express.js

                                    sudo npm install --save express


Code Hello World!

1. In the shell, create an server.js file with the following command

                                    sudo vim server.js


2. Insert the following codes

                                    const express = require('express');
const app = express();
app.get('/', (req, res) => {
   res.send('Hello World!');});


Start Node Server

1. Start the server with the command

                                    node app.js


2. Connect to it by clicking the VM's IP address and appending port 3000


3. If IP address doesn't open, you can copy the IP and paste it to the web browser


4. Make sure to add Port 3000 after the IP adress

                                    :3000


5. You see a response from the web broswer

                                    Hello World!
