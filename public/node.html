Node Tutorial

In this tutorial we will be installing Node.js, we want Node.js on our google cloud program. In order to have Node.Js on our google cloud virtual machine we need to install node, then install node package management at which point will need to install NPM and do a final check to see if the package manager is running!


Install Node

1. First get all the new update for your VM

                                sudo apt-get update


2. Install Nodejs

                                sudo apt-get install -y nodejs


3. Install Node Package Management

                                sudo apt-get install npm


4. Check if Node is running 

                                node -v


5. Check if Node Package Management is running

                                npm -v
