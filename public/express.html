Express Tutorial

In this tutorial, we 're going to install Expressjs for the web application. We 're going to need to initialize npm and start scripting the.json package files. Then we can move on to install Express on the Virtual Machine and save it there!


Install Express

1. Initialize a package.json file with the following command

                                     sudo npm init


2. Add a start script to your package.json file

                                    "script":{ 
"start": "app.js"


3. Install Express.js

                                    sudo npm install --save express
